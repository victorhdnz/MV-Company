# Assets do Lanyard

Para que o efeito Lanyard funcione corretamente, você precisa adicionar os seguintes arquivos nesta pasta:

1. **card.glb** - Modelo 3D do cartão
2. **lanyard.png** - Textura da corda do lanyard

## Como obter os assets

Os assets podem ser baixados do repositório oficial do Lanyard:

1. Acesse: https://github.com/21st-dev/lanyard
2. Navegue até a pasta `src/assets/lanyard`
3. Baixe os arquivos:
   - `card.glb`
   - `lanyard.png`
4. Coloque os arquivos nesta pasta: `public/assets/lanyard/`

## Estrutura de pastas

```
public/
└── assets/
    └── lanyard/
        ├── card.glb
        ├── lanyard.png
        └── README.md (este arquivo)
```

## Personalização

Você pode personalizar o modelo 3D do cartão usando o editor online:
- https://modelviewer.dev/editor/

E editar a textura da corda usando qualquer editor de imagens.
